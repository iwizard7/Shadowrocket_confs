# Shadowrocket Configurations for Russia (ULTRA Edition 2026)

🚀 **Самый продвинутый набор оптимизированных конфигураций Shadowrocket для бесшовного интернета на macOS и iOS в условиях 2026 года.**

## Главные фишки версии ULTRA

В этой версии мы выжали максимум из возможностей Shadowrocket для работы в сложной сетевой среде.

### 🎧 Discord Voice Fix
- Решена проблема с обрывами и тишиной в голосовых каналах Discord. Специфический UDP трафик направляется через прокси по выделенным правилам.

### ⚡ DNS-over-QUIC
- Использование защищенного и сверхбыстрого протокола `quic://dns.adguard.com`. Минимальная задержка (latency) при резолве имен даже на нестабильном 4G/5G.

### 🎮 Мастер Гейминга (Steam/Epic/4PDA)
- **Умное разделение**: Игровой трафик и загрузки идут напрямую (`DIRECT`) для нулевого пинга, а чаты, сообщества и заблокированные порталы — через прокси.

### 🚀 Технологический стек
- **[Host] Mapping**: Высокоскоростной маппинг доменов (Google и др.) для мгновенного доступа.
- **[URL Rewrite]**: Автоматическая очистка ссылок от трекеров и редиректы с заблокированных локальных версий сайтов на глобальные.
- **[Script] Ready**: Конфиг подготовлен для работы со скриптами (отображение статуса подписки, баланса трафика).

## Базовые возможности

### 🛡️ Блокировка и защита
- **AdBlock**: Полное вырезание рекламы, баннеров и аналитики (Rule-Sets от blackmatrix7).
- **Security First**: DoH/DoT шифрование всех DNS-запросов.

### 🌎 Сервисы и Соцсети
- **YouTube & Streaming**: Полная разблокировка без троттлинга.
- **AI Tools**: Приоритетный доступ к OpenAI (ChatGPT), Claude, DeepL.
- **Apple & Media Bypass**: Прямой доступ к сервисам Apple, VK Video и национальному мессенджеру **MAX**.

## Структура репозитория

- `/MacOS/macos_ultra.conf` — ULTRA-версия для macOS.
- `/iOS/ios_ultra.conf` — ULTRA-версия для iPhone/iPad.

## Как установить

1. Скопируйте ссылку на RAW-файл нужного конфига.
2. В Shadowrocket: **Config** -> **«+»** -> **Download from URL**.
3. Вставьте ссылку на RAW-файл (например, `.../MacOS/macos_ultra.conf`).
4. Установите **Global Routing** в режим **Config**.
4. (Опционально) Для работы URL Rewrite установите и включите CA-сертификат в разделе **HTTPS Decryption**.
