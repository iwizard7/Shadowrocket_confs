# Shadowrocket Configurations for Russia (Optimized)

🚀 **Персональный набор оптимизированных конфигураций Shadowrocket для комфортной работы в России на macOS и iOS.**

## Основные возможности

Данные конфигурации собраны на базе популярных решений от сообщества (misha-tgshv, blackmatrix7) и дополнены специфическими правилами для максимальной производительности.

### 🛡️ Блокировка и защита
- **AdBlock**: Интегрированная блокировка рекламы, баннеров и трекеров (Rule-Sets от blackmatrix7).
- **DNS Security**: Использование DoH/DoT (Cloudflare, Google) для защиты от перехвата трафика и DNS-отравления.

### 🌍 Навигация и обход блокировок
- **YouTube & Social**: Полная поддержка YouTube (без троттлинга), Instagram, Facebook, X (Twitter).
- **AI Services**: Выделенные правила для стабильного доступа к OpenAI (ChatGPT), Claude (Anthropic), DeepL и другим ИИ-сервисам.
- **Messenger Support**: Поддержка голосовых вызовов в Telegram и WhatsApp через оптимизированные порты.

### ⚡ Оптимизация для РФ
- **Apple Direct**: Весь трафик системных сервисов Apple (iCloud, App Store, обновления, авторизация) идет напрямую для максимальной скорости и стабильности.
- **VK Video & Media**: Прямое соединение для VK Video, Mail.ru и других крупных российских медиа-ресурсов.
- **MAX Messenger (VK)**: Добавлена полная поддержка и байпас для нового национального мессенджера MAX.
- **GEOIP Bypass**: Автоматическое распознавание российских IP-адресов и их направление мимо прокси (автопилот).

## Структура репозитория

- `/MacOS/sr_ru_basic.conf` — Специально оптимизированная версия для macOS (улучшенная работа с системными сервисами).
- `/iOS/sr_ru_basic.conf` — Оптимизированная версия для iPhone и iPad с учетом мобильных особенностей.

## Инструкция по установке

1. Откройте **Shadowrocket** на вашем устройстве.
2. Перейдите на вкладку **Config**.
3. Нажмите кнопку **«+»** (или "Download from URL").
4. Вставьте ссылку на RAW-файл из этого репозитория (например, `.../MacOS/sr_ru_basic.conf`).
5. В настройках Shadowrocket установите **Global Routing** в режим **Config**.
6. Наслаждайтесь свободным интернетом!
